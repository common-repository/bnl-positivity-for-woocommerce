== Description ==

Aggiunge fra i metodi di pagamento con gateway attivabili in WooCommerce il circuito BNL Positivity

== Installation ==

1. Install the plugin from Wordpress Plugin Repository or download from the plugin page and upload `bnl-positivity-for-woocommerce` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WooCommerce -> Settings -> Checkout tab and click BNL Positivity
